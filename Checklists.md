## LOSS (LetsAI) — Pre / During / Post (short)

Pre:

* Finalize topic/speaker/moderator/date
* Collect bio/photo/abstract & consent
* Speaker Packet + dry run
* Luma event + Zoom link + Canva banner + QR
* Promo 7–10 days prior; reminders at T–3, T–1, T–0

During:

* Start 15–20 min early; assign Host/Co-host/Moderator/Recorder
* Record session; share Code of Conduct
* Capture screenshot for socials; announce next event

Post:

* Upload to YouTube (per recording policy)
* Post recap: LinkedIn + Medium
* Update `/events/` and metrics

---

## ZOWS (Zero2Hero Workshop) — Pre / During / Post (short)

Pre:

* Confirm venue, capacity, A/V, snacks
* PAIC standee & registration desk
* Test devices; backup laptop + hotspot
* Signages + first-aid + emergency contacts
* Create listings; promo 10–14 days prior

During:

* Volunteer briefing (roles)
* Check-in flow (QR/manual)
* Record segments; take photos/clips
* Thank sponsor on slides

Post:

* Recap post + group photo
* Upload edited recordings; file receipts
* Update budget tracker; archive materials

## Hybrid event (must-do)
